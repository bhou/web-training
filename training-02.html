Selector

Selector Types

  • Attribute selector

  • Pseudo-elements
  • Combinators
  • Multiple selectors

Pseudo-classes

A CSS pseudo-class is a keyword, preceded by a colon (:), added to the end of selectors to specify you want to style the selected elements, and only when they are in certain state.

Element state

  • visited
  • link
  • hover
  • first-child
  • ...

a full list of element state can be found here

Pseudo-classes Example

<div id="hover-example">Hover Here</div>
#hover-example:hover {
	color: red;
}
Hover Here

Pseudo-elements

Pseudo-elements are keywords, preceded by two colons (::), that can be added to the end of selectors to select a certain part of an element.

Pseudo-elements Example

<div id="selection-example">Select the text here</div>
#selection-example::selection {
	color: red;
}
Select the text here

Other Pseudo-elements

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::backdrop

Note: Different browsers might uses a different name for these elements

Combine Multiple Selectors Together

Multiple Selectors

SyntaxDescription
A, BElements that match A or B
A BElements that match B, which is a descendant of element matching A
A > BElements that match B, which is a direct child of element matching A
A + BElements that match B, which is the next sibling of element matching A
A ~ BElements that match B, which is one of the next sibling of element matching A
