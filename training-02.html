action: set style

CSS Rule

If selector then declaration

	declaration;
	declaration;
	declaration;	
}

Selector

Selector

In CSS, selectors are used to target the HTML elements on our web pages that we want to style

Selector Types

  • Simple selector
  • Attribute selector
  • Pseudo-classes
  • Pseudo-elements
  • Combinators
  • Multiple selectors

Simple selector

Select elements by element type, id or class

<div id="simple-selector-example" class="code">...<div>
div {
...
}

#simple-selector-example {
...
}

.code {
...
}

Attribute selector

Select elements by element attribute

<div favourite-fruit="apple" attr2="value2">...</div>
<div favourite-fruit="apple banana kiwi">...</div>
[favourite-fruit] {
...
}

[favourite-fruit="apple"] {
...
}

[favourite-fruit~="apple"] {
...
}
