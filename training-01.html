Semantic Element

General Block Element: <div>

<div> is the generic container for flow content. It has no effect on the content or layout until styled using CSS

General Inline Element: <span>

<span> is a generic inline container for phrasing content, which does not inherently represent anything

You can build your whole webpage with div and span
