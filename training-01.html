Semantic Element

General Block Element: <div>

<div> is the generic container for flow content. It has no effect on the content or layout until styled using CSS

General Inline Element: <span>

<span> is a generic inline container for phrasing content, which does not inherently represent anything

You can build your whole webpage with div and span

Form

What newspaper can't do: send data to the author

What is HTML form ?

HTML Forms are one of the main points of interaction between a user and a web site or application. They allow users to send data to the web site.

Form Example

<form action="/url-for-action" method="post">
	<div>
		<label for="name">Name:</label>
		<input type="text" id="name" name="user_name">
	</div>
	<div>
		<label for="mail">E-mail:</label>
		<input type="email" id="mail" name="user_mail">
	</div>
	<div>
		<label for="msg">Message:</label>
		<textarea id="msg" name="user_message"></textarea>
	</div>
	<div class="button">
		<button type="submit">Send your message</button>
	</div>
</form>
